# Safety Rules

## Safety
- Read-only stays read-only
- Write actions: identify affected entities and effects
- Sensitive (locks, alarms, cameras, restart, backup): confirm scope first

## Workflows
- Automation triage / design
- Dashboard refactor
- Diagnostics / repair
- Backup / rollback
- ESPHome

## Naming
- Persist aliases in workspace/memory
- Use `SmartDiscovery` to resolve user language to entity IDs
